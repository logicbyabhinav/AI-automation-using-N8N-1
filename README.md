# AI-Automation-using-N8N-2

> An intelligent client intake and project management automation system built with **n8n**, powered by an **AI Agent** as the core workflow brain.

---

##  Project Overview

This project automates the entire client onboarding process for a **software development agency website**. When a potential client fills out a form on the website, n8n kicks in to intelligently process their data, decode their requirements using an AI agent, send a confirmation email, and log everything into a Google Sheets database — all without manual intervention.

---

##  How It Works

### 1. Client Fills Out the Website Form
The website collects the following information:

**Personal / Contact Details**
- Full Name
- Address
- Occupation
- Email Address
- Phone Number (optional)

**Project / Website Requirements**
- Type of website or software needed
- Core features and functionality
- Target audience
- Design preferences (if any)
- Deadline / Timeline

**Budget & Team Preferences**
- Estimated budget range
- Developer team size preference (solo dev, small team, full team)
- Any specific tech stack requirements
- Additional conditions or notes

---

### 2. n8n Workflow Takes Over

Workflow Overview
```
Webhook Trigger (Form Submission)
        |
Data Formatting Node
(Cleans and structures raw form input)
        |
 AI Agent — Core Brain
(Interprets requirements, generates summary & ticket ID)
        |
        |--- Google Sheets Node
        |    (Logs structured client data as a new row)
        |
        `--- Gmail / SMTP Node
             (Sends confirmation email to client)
```
Nodes Breakdown
NodeTypeRoleWebhook / Form TriggerTriggerReceives raw form data on submissionCode / Set NodeData PrepCleans, formats, and maps incoming fieldsAI AgentIntelligenceParses project requirements, generates a human-readable summary, creates a unique Ticket IDGoogle SheetsOutputAppends a fully structured row to the client databaseGmail / SMTPOutputSends the client a confirmation email with their decoded requirements and next steps
### AI Agent — The Brain

The **AI Agent node** is the intelligence layer of this workflow. It receives the structured client data and is responsible for:

- Reading and **interpreting** the client's raw project description and requirements
- Generating a **clean, professional summary** of what the client wants built
- Producing a **unique Ticket ID** to reference the inquiry
- Flagging any **missing or ambiguous fields** for review
- Preparing the final content used in both the **confirmation email** and the **Google Sheets log**

### Email Confirmation Includes

- Client's personal details (as submitted)
- AI-generated summary of their project requirements
- Budget and team conditions recap
- Auto-generated Ticket ID for future communication
- Next steps / what to expect

---

### 3. Google Sheets Database

All client data is saved in a structured Google Sheet with the following columns:

| Column | Description |
|--------|-------------|
| Timestamp | Date and time of submission |
| Client Name | Full name |
| Email | Contact email |
| Occupation | Client's occupation/industry |
| Project Type | Website or software type |
| Requirements Summary | AI-decoded summary |
| Budget Range | Submitted budget |
| Team Preference | Developer team conditions |
| Status | Default: `New` |

---

### 4. Confirmation Email

The client receives an automated email containing:
- A greeting with their name
- A structured summary of their submitted project details
- Their decoded requirements (generated by the AI agent)
- Next steps / what to expect

## Getting Started

### Prerequisites
- [n8n](https://n8n.io/) installed (self-hosted or cloud)
- Google account with Sheets API enabled
- SMTP credentials or Gmail account for sending emails

### Setup Steps

1. **Clone this repository**
   ```bash
   git clone https://github.com/logicbyabhinav/AI-automation-using-N8N-2.git
   ```

2. **Import the workflow**  
   Open n8n → Import from file → select `workflow/n8n-workflow.json`

3. **Configure credentials in n8n**
   - Add your Google Sheets OAuth credentials
   - Add your Gmail / SMTP email credentials
   - Set your AI model API key (OpenAI / Anthropic / etc.)

4. **Set up the webhook URL**  
   Copy the webhook URL from n8n and add it to the frontend form's action.

5. **Create your Google Sheet**  
   Set up a sheet matching the column structure above and link it in the workflow.

6. **Test the flow**  
   Fill out the form and verify that data lands in Google Sheets and the email is received.
